591 房屋租賃資料查詢服務 v2.0.0

提供強大的房屋搜尋與統計功能 | 🔒 個資保護機制 | 📊 實時資料分析

🎯 API 功能概覽

🔍 智能搜尋

支援多條件組合搜尋：房屋類型、價格範圍、發布者類型、關鍵字等

📊 資料統計

提供完整的市場統計分析，包含價格覆蓋率、發布者分析等

🔒 隱私保護

手機號碼智能遮罩處理，保護個人隱私資訊

⚡ 高效能

Redis 快取機制 + BigQuery 最佳化，提供毫秒級回應

1️⃣ 基本搜尋

GET /api/properties?limit=5

取得前5筆房屋資料，適合初始載入或測試連接

2️⃣ 條件搜尋

GET /api/properties?house_type_id=2&min_price=15000&max_price=40000

搜尋電梯大樓，價格範圍 15,000-40,000 元

3️⃣ 關鍵字搜尋

GET /api/properties?keyword=烏日&limit=10

搜尋包含「烏日」關鍵字的物件

4️⃣ 分頁查詢

GET /api/properties?limit=20&offset=40

取得第3頁資料（每頁20筆，跳過前40筆）

GET /api/properties - 房屋物件搜尋

支援參數：house_type_id, poster_type_id, min_price, max_price, keyword, limit, offset

GET /api/statistics - 整體統計資料

提供總物件數、價格分析、發布者分布等統計資訊

GET /api/house-types - 房屋類型列表

取得所有房屋類型的 ID 和名稱對應表

GET /api/poster-types - 發布者類型列表

取得所有發布者類型的 ID 和名稱對應表

GET /health - 健康檢查

檢查 API 服務狀態和相關依賴服務

🎯 搜尋最佳化

  • 分頁查詢：建議每次最多查詢 50 筆資料以獲得最佳效能
  • 條件組合：可同時使用多個篩選條件，例如：房屋類型 + 價格範圍 + 關鍵字
  • 快取利用：相同查詢條件會自動使用快取，提供更快回應速度

🔒 資料安全

  • 手機遮罩：API 自動對手機號碼進行遮罩處理（例：09****123）
  • 個資保護：符合個資法規範，保護使用者隱私

📊 回應格式

標準成功回應： 
{
  "success": true,
  "data": [...],
  "count": 10,
  "from_cache": false
}

常見錯誤類型

  • 500 Internal Server Error：伺服器內部錯誤，請檢查參數格式
  • 參數錯誤：請確認 house_type_id (1-5) 和 poster_type_id (0-2) 範圍
  • 查詢逾時：建議減少查詢筆數或縮小查詢範圍

除錯建議

  • 先使用 /health 端點檢查服務狀態
  • 使用基本查詢 /api/properties?limit=1 測試連接
  • 檢查 Console 中的錯誤訊息
🛡️ 隱私保護說明： 為保護用戶隱私，本 API 對手機號碼進行智能遮罩處理。例如： 0912345678 → 09****678， 04-22334455 → 04****455

🚀 一鍵測試連結

🏠 基本搜尋 (5筆) 🏢 電梯大樓 👤 房東物件 💰 價格篩選 🔍 關鍵字搜尋 📊 統計資料 ❤️ 健康檢查
📋 資料映射參考：
🏠 房屋類型 (house_type_id): 1=透天厝, 2=電梯大樓, 3=公寓, 4=別墅, 5=其他(停車位)
👥 發布者類型 (poster_type_id): 0=房東, 1=房仲, 2=代理人
💰 價格範圍： 建議使用 15,000-50,000 進行測試
